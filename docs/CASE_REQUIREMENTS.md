# Соответствие основным задачам кейса

| Задача | Реализация в репозитории |
|--------|---------------------------|
| **RAG:** загрузка PDF, DOCX, TXT и вопросы по содержанию | `document-service` + `rag-service` (`ingest`, чанки в PostgreSQL, TF-IDF / опционально эмбеддер). Вопросы: вкладка **Чат** в `/workspace/:id` и страница **`/chat`** по всем готовым документам. |
| **Чат с источниками** | `ragQuery` → контекст в системный промпт → `aiChat`. Блок **«Источники»** с фрагментами и оценкой релевантности. |
| **Аудиопересказ** | Сценарий диалога двух ведущих по тексту документа; выбор **тона** (научный / популярный) и **темпа** речи (медленно / нормально / быстро) в UI; кнопка **озвучить** через Web Speech API в браузере (демо без отдельного TTS-сервера). |
| **Формализованный отчёт** | Вкладка **«Отчёт»** в workspace: резюме в **официальном стиле по фиксированному шаблону** (цель, содержание, положения, выводы, рекомендации). |
| **Mindmap** | Вкладка **«Mindmap»**: разбор иерархии из ответа модели и **интерактивный граф** (узлы и связи, подсветка по наведению/клику). |
| **STT (опционально)** | Кнопка **🎤** в чатах: загрузка аудио → `POST /api/v1/ai/transcribe` → внешний **POST /v1/audio/transcriptions** (`STT_BASE_URL`, напр. порт **6640**). |
| **Таблица из текста (опционально)** | Вкладка **«Таблица»** в `/workspace/:id`: контекст из RAG → `POST /api/v1/ai/extract-table` → CSV (скачивание, открытие в Excel). |

Дополнительно (не из списка обязательных, но есть в продукте): тесты, карточки, краткое содержание и темы на главной вкладке документа.
