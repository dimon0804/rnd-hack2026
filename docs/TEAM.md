# Команда и зоны ответственности

## Tech Lead (ты)

- Архитектура, инфраструктура (`docker-compose`, Nginx, стандарты API).
- **api-gateway** — единая точка входа, прокси, CORS, rate limit.
- **auth-service** — JWT, безопасность входа.
- **rag-service** — пайплайн embeddings, векторное хранилище, контракты с AI.
- Координация интеграции, ревью PR Никиты, мерж в `main` по согласованию.

## Никита (developer)

### Backend

- **document-service** — загрузка и обработка файлов (PDF, DOCX, TXT), валидация, хранение.
- **generation-service** — саммари, тесты, карточки, mindmap (и опционально подкаст).
- **ai-service** — частично: реализация вызовов Mistral, эндпоинты по контрактам с Tech Lead (промпты/схемы согласовываются).

### Frontend

- **UI/UX** — визуальный слой, адаптив, анимации в меру.
- Страницы:
  - загрузка файлов;
  - чат (RAG);
  - экран результатов генерации (саммари, тесты, карточки и т.д.).

## Взаимодействие

- Контракты REST/OpenAPI — в `develop` до подключения фронта к новым эндпоинтам.
- Вопросы по Mistral/промптам — через Tech Lead или короткий sync.
